# Configuração de problemas

A variável *Configuração de problemas* O Caminho de aprendizagem do Workfront One agora está mapeado para os tutoriais do Experience League Workfront recém-migrados e otimizados. Todas as mesmas informações são capturadas nos novos tutoriais. Navegue até o novo Experience League [Workfront Tutorials](https://experienceleague.adobe.com/docs/workfront-learn/tutorials-workfront/home.html)e vá para a página *Configurar padrões do sistema* na seção *Administração e configuração* na navegação à esquerda da página Tutorials.
